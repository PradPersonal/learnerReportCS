# 1. Containerize Your Applications
  - Create a Dockerfile for both your frontend and backend applications. This defines how to build the container images. 
  -  - For Backend & Frontend: Navigate to each service's directory and run docker build -t your_dockerhub_username/service-name:tag ..
  -  - Push Images: Log in to your container registry and push the images: docker push your_dockerhub_username/service-name:tag.
